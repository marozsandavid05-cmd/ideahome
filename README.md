# IdeaHome — weboldal

Statikus weboldal (HTML + CSS + vanilla JS, build-lépés nélkül).
Lakberendezés, belsőépítészet és home staging.

## Megtekintés
Nyisd meg az `index.html`-t böngészőben, vagy nézd élőben GitHub Pages-en.

## Szerkezet
- `index.html` + a többi `*.html` — főoldal és aloldalak
- `munkaink/` — projekt-eset oldalak
- `assets/css/site.css` — design rendszer (aktív)
- `assets/js/site.js` — interakciók
- `assets/img/` — képek (webp); `portfolio/opt/proj/<projekt>/` a galériák

## GitHub Pages bekapcsolása
Repo → **Settings → Pages → Source: Deploy from branch → `main` / `(root)` → Save**.
Pár perc múlva él: `https://<felhasznalonev>.github.io/<repo>/`
